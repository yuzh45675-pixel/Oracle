# Tarot Oracle · 塔罗与雷诺曼占卜

维特塔罗（78 张）+ 雷诺曼（36 张）在线占卜 Web 应用。

## 本地运行

```bash
npm install
npm run dev
```

浏览器打开 [http://localhost:3000](http://localhost:3000)

## 部署到公网（国内可访问）

请阅读 **[DEPLOY.md](./DEPLOY.md)**，按步骤把项目上传到 GitHub，再用 Zeabur 发布网站。

## 技术栈

- Next.js · React · TypeScript · Tailwind CSS · Framer Motion
